# Advanced Notes App

A simple notes app built with HTML, CSS, and JavaScript. It lets you add, search, and manage notes in a clean interface.

## Features

- Add new notes with a title and content
- Search notes easily
- Character count display
- Dark mode toggle
- Clean and modern layout

## Built With

- HTML
- CSS
- JavaScript

## How It Works

The app uses a form to add notes and shows them in the notes section.  
You can type in the search box to quickly find a note.  
The theme button switches between light and dark mode for a better user experience.

- Updating the UI dynamically
